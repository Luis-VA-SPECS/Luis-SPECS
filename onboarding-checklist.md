# Client Onboarding Checklist

*Run through this for every new client to make sure nothing gets missed.*

## Before Kickoff
- [ ] Client intake form completed
- [ ] Rate and payment terms agreed in writing
- [ ] Contract/agreement signed (if applicable)
- [ ] First invoice schedule set

## Access Setup
- [ ] Access granted to required tools/platforms
- [ ] Login credentials received via secure method
- [ ] Added to relevant communication channels (Slack, email, etc.)
- [ ] Shared drive/folder access confirmed

## Workflow Setup
- [ ] Client added to Task Tracker
- [ ] Client added to Client CRM sheet
- [ ] Recurring check-in scheduled (day/time agreed)
- [ ] Reporting format/frequency confirmed

## First Week
- [ ] First task/project scoped and started
- [ ] Initial check-in completed
- [ ] Any tool/access issues resolved
- [ ] Client expectations re-confirmed after first deliverable
