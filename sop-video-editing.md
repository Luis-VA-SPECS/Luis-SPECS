# SOP: Video Editing Workflow

*Example process doc — shows clients you have a repeatable, reliable workflow. Duplicate this file for other repeatable tasks (e.g., social posting, inbox triage).*

## 1. Receive Brief
- Confirm the Video Editing Brief is filled out completely
- Download/access raw footage from provided link
- Flag any missing assets to the client immediately

## 2. Rough Cut
- Import footage into editor (CapCut / Premiere Pro / DaVinci Resolve)
- Select best takes and assemble rough sequence
- Match to target length from the brief

## 3. Refine
- Add transitions, captions, and color correction
- Balance audio levels
- Add intro/outro branding if required

## 4. Internal Review
- Watch full video back before sending
- Check against the "Checklist Before Sending Final Cut" in the video brief template

## 5. Client Review
- Send draft with clear version naming (client_project_date_v1)
- Log feedback and revision requests
- Apply revisions (track how many rounds used vs. included in scope)

## 6. Final Delivery
- Export in correct format/resolution for target platform
- Deliver via agreed method
- Update Task Tracker status to "Done"
- Log hours in Timesheet
